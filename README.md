# Teste Java

Este repositório contém vários programas Java simples para testes básicos.

## Arquivos e Explicações

### Teste1.java
Verifica se um número informado pelo usuário é par ou ímpar.

### teste2.java
Conta de 1 até um número informado pelo usuário, imprimindo cada número.

### Teste3.java
Pede ao usuário para informar quantos números deseja digitar e encontra o maior entre eles.

### teste4.java
Conta quantas vogais (a, e, i, o, u) existem em uma palavra informada pelo usuário.

### teste5.java
Compara dois números informados e retorna o maior usando uma função.

### teste6.java
Calcula o fatorial de um número informado usando uma função recursiva (na verdade, iterativa).

### teste7.java
Pede uma quantidade de números e conta quantos deles são pares.

### teste8.java
Verifica se uma palavra informada é um palíndromo (lê-se igual de trás para frente).
